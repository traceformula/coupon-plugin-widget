- Setup and config the plugin:

Step 1: Copy plugin to wordpress plugins in folder.

Step 2: Edit the config.xml in plugin folder.
    - Edit the value in fileName node. The plugin will save xlsx file as this name.
    - Edit the value in downloadUrl node. The plugin will download xlsx file in this url.
    - Edit the value in scheduleTime node. The plugin will download file in schedule of this time (second).

Step 3: Active the plugin.
